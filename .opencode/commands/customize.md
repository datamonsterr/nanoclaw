Run the NanoClaw customize skill. Use this when you want to:

- Add a new messaging channel (Telegram, Slack, Discord, Gmail)
- Add integrations (Ollama, PDF reader, image vision, voice transcription)
- Change assistant name or trigger pattern
- Modify routing or message handling
- Add or change scheduled tasks

This is an interactive skill that asks questions about what you want to customize.
